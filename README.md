# violencia_intrafamiliar
El archivo "[Violencia_intrafamiliar].ipynb" es un notebook de Jupyter que analiza datos sobre solicitudes de medidas de protección por violencia intrafamiliar en Medellín, Colombia, desde 2003 hasta 2023. Contiene 153 celdas, alternando entre texto explicativo en Markdown y código en Python. Incluye importación de bibliotecas (como pandas, matplotlib y seaborn), carga del dataset CSV, exploración de datos, visualizaciones y análisis estadístico. Ninguna celda ha sido ejecutada aún. El enfoque es científico-político, evaluando tendencias y patrones para informar políticas públicas.
